# Grails Spring Security Core Plugin Custom Authentication

This repository contains the worked sample app for the [Grails Spring Security Core Plugin Custom Authentication](https://grails.apache.org/guides/grails-spring-security-core-plugin-custom-authentication/4/guide/index.html) guide on the Apache Grails site.

## Run the example

`./gradlew bootRun`

## Maintained at apache/grails-static-website

The guide narrative is maintained at https://github.com/apache/grails-static-website. Changes to the guide content go there; this repo holds the worked sample app only.
